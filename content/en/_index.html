{{< /blocks/cover >}} {{% blocks/lead color="dark" %}} This site provides explanation to use kaptainwutax's libraries. The first thing you should do is create a new project in a folder by doing `gradle init`. Then you add those lines in your build.gradle:
```groovy repositories { mavenCentral() maven { url "https://jitpack.io" } } dependencies { implementation 'com.github.KaptainWutax:SEED:c540b5224dbb4dc80fe8a25bfcfcf9383f635a0c' } ```
